# Task Manager

A minimal CLI task manager — simple, fast, and easy to extend.

## Project Files

- `cli.js` — Command-line interface entrypoint
- `taskmanager.js` — Core task operations (add, list, complete)
- `package.json` — Dependencies and scripts

## Commands

- **`add`** — Create a new task. Choose type (`work`, `personal`, or `general`), add a title and due date.
- **`list`** — Display all tasks with details.
- **`complete <title>`** — Mark a task as done.
- **`remove <title>`** — Delete a task.
- **`quit` / `exit`** — Exit the CLI.

**Note:** Tasks are stored in memory only (no file persistence). Restarting clears the list.

## Quick Start

```bash
npm install
node cli.js
```
